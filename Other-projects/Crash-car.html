Crash-car

A light weight game that was developed using the GLUT openGL library

The game was developed with exapnsion in mind that’s why you would find that our game is divided into three main components

Game Logic

Power-up class

Cars class

The game already has 2 power-ups built in it 2x score and invincibility so make sure you take them and score the highest

Also if you want to add your own custom power up u can just add into the power up class and you’re good to go
